Minecraft Reward System

An Arduino-powered homework accountability system

Arduino Uno 16x2 LCD Joystick EEPROM v5.0
Features Scoring LCD Preview Wiring Setup Changelog Download

✨ Features

🎯

Anti-Cheat Quiz

50 questions across 5 categories. Arduino randomly picks 1 per category giving 100,000 possible combinations.

💾

Permanent Stats

Stats saved to EEPROM. Survives power off. Hold button 3 seconds on Stats page to reset.

🕹

Swipe Navigation

Push joystick left or right to browse between Homework, Stats and Free Timer pages.

⏱

Free Timer

Set any custom countdown from 1 to 60 minutes using the joystick. No quiz needed.

🔲

Visual UI

Bordered page selection, triangle navigation indicators and score feedback on the LCD.

⚡

5 Session System

Complete 5 quiz sessions and all your earned time combines into one big Minecraft countdown!

🚀 Setup Guide

1

Download the code

Download minecraft_reward_system.ino from this repo

2

Install library

In Arduino IDE go to Library Manager and install LiquidCrystal_I2C by Frank de Brabander

3

Wire it up

Connect LCD and Joystick according to the wiring table above

4

Upload

Open the file in Arduino IDE or ArduinoDroid, select your board and upload!

5

Play!

Complete 5 homework quiz sessions to earn your Minecraft time 🎮
